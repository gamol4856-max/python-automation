# CSV to JSON Converter (Python)

This is a simple Python script that converts data from a CSV file into JSON format.

I created this project to practice:
- File handling in Python
- Reading CSV files using `csv.DictReader`
- Converting rows into dictionaries
- Writing structured data to JSON
- Basic Git and GitHub workflow

## How It Works
- Reads data from `input.csv`
- Converts each row into a dictionary
- Stores all rows in a list
- Writes the list into `out.json`

## Project Files
- `convert.py` – Main Python script
- `input.csv` – Sample CSV input file
- `out.json` – Generated JSON output
- `.gitignore` – Ignored files and folders
